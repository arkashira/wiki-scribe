# Wiki Scribe

A Python project for monitoring auto-approved pages.

## Usage

1. Create a `WikiScribe` object with a list of `AutoApprovedPage` objects.
2. Use the `get_monthly_trend` method to get the monthly trend of auto-approved pages.
3. Use the `refresh_data` method to refresh the data.
4. Use the `filter_by_project_and_date_range` method to filter the data by project and date range.

## Testing

Run the tests using `pytest`.
