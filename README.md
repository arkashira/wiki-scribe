# Wiki Scribe
A tool for managing wiki drafts and sending notifications to maintainers.

## Usage
1. Create a new instance of the `WikiScribe` class.
2. Add drafts to the wiki scribe using the `add_draft` method.
3. Get drafts needing review using the `get_drafts_needing_review` method.
4. Approve or reject drafts using the `approve_draft` or `reject_draft` methods.

## Testing
Run the tests using `pytest` to ensure the code is working correctly.
