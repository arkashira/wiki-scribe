# Wiki Scribe
A Python project for managing timelines.

## Usage
1. Create a JSON file with the timeline data.
2. Run the `load_timeline` function to load the timeline data.
3. Run the `validate_timeline` function to validate the timeline data.
4. Run the `store_timeline` function to store the timeline data.
5. Run the `display_timeline` function to display the timeline data.

## Testing
Run the tests using `pytest`.
